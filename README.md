## Descripcion Del Proyecto
Este es el proyecto Evaluacion Final en cual se ayuda a la empresa a digitalizar tours como clientes, provedor y guias

## Estructura General Del Proyecto

```plaintext
📁 Resources
├── Tours.txt #Tipo de texto donde estan los tours agregados
├── Personas.txt #Tipo de texto donde estan las personas agregadas
📁 src/
````

## Instrucciones para ejecutar el proyecto

1. Clonar el repositorio desde Github.

```bash
git clone https://github.com/HectorSotoDuocuc/LlanquihueTourEFTHectorSoto.git
```

2. abre el proyecto en IntelliJ IDEA.

3. Ejecutar el archivo `Main.java` desde el paquete `ui` buscando el boton verde de play que estara arriba a la derecha.

4. Saldra un menu donde podra agregar panoramas, eliminarlos, buscar por nombre o mostrar todos ademas se agrega un interfaz para poder agregar ruta gastronomica guias y provedores.
